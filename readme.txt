1  致使用者语
      这是一个数学函数制图的专业软件，我们的idea来自于window下的winplot软件，它能完成各种函数的绘制功能。在linux中，已有函数制图软件有Grace和kmplot等，其中kmplot制作相对全面、美观一些，但都只能支持2D的函数制图。我们制作的这个软件，已经能支持3D函数作图，而且支持3种不同的3D模式。
   
   你们的认同是我们最大的欣慰。^_^

   优点：
	1.少有的linux上3D制图的非编程软件
	2.有一定技术含量，使用较为方便
	3.完全原创


2  程序运行须知
     需要依赖：
该程序除了gtk等常用依赖外，还用到了R和rpy2作为依赖，相应的依赖可参照以下代码安装

更新	sudo apt-get update
安装R	sudo apt-get install r-base r-base-dev
终端打开	R
       R中输入	install.packages("emdbook")
       选择相应镜像路径安装好emdbook包即可（若安装不成功，导致软件运行错误，请参照3 一些说明中的第2条）
安装rpy2	sudo apt-get install python-rpy2

4 使用帮助
                1、本产品支持三角、反三角函数，对数函数，幂指函数。
		2、绘图范围旁边的旋转按钮点击改变范围后函数可以动态生成图像。
		3、改变参数值后按回车可以自动执行。
		4、2D显示作图中未知数只识别x。
		5、2D隐式作图和3D作图中未知数只识别x,y。
		6、3D作图有默认图形，3D图形和等高线图三种格式。
		7、点击旋转按钮改变绘图范围会有不一样的效果。
		8、乘法和除法必须用符号*和/。
		9、ln用log代替,其他对数函数格式为logN(x)(其中N为常数)。
		10、反三角函数的格式为：asin(x),acos(x),atan(x)。
		11、圆周率用pi表示。
		12、正负号可以直接识别，正数前面的+号可以不省略。
